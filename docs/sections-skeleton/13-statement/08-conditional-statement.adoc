=== Conditional statements

// 716
The conditional statement uses standard syntax and semantics familiar from many
programming languages.

// 717
However, the condition expression in P4 is required to be a Boolean
(and not an integer).

// 718 - Modified
${syntax: conditionalStatement}

// 719
When several `if` statements are nested, the `else` applies to the innermost
`if` statement that does not have an `else` statement.
