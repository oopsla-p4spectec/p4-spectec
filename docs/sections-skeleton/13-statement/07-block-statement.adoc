[#sec-block-statement]
=== Block statements

// 704
A block statement is denoted by curly braces. It contains a sequence of
statements and declarations, which are executed sequentially. The declarations
(e.g., variables and constants) within a block statement are only visible
within the block.

// 705 - Modified
${syntax: blockStatement blockElementStatementList blockElementStatement}
