[#sec-continue-statement]
=== Continue statements

// Added
A `continue;` statement skips the current iteration of the enclosing `for`
loop.

// Added
${syntax: continueStatement}

==== Type checking

// Added
After type checking, a `continue;` statement has the form:

// Added
${syntax: continueStatementIR}

// Added
${rulegroup-source: Stmt_ok/continueStatement}
${rulegroup-prose: Stmt_ok/continueStatement}

==== Compile-time evaluation
