=== Cast insertion

// Added
When typing a P4~16~ program to a P4~IR~ program, all implicit casts are made
explicit. This process is called _cast insertion_. A cast may be inserted
such that an expression matches the expected type, and it may also be
inserted to ensure that two expressions have the same type.
