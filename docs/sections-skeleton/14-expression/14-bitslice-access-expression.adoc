[#sec-bitslice-access-expression]
=== Bitslice access expressions

// Added
A bitslice access expression is used to access a contiguous range of bits
within a numeric value. The syntax for a bitslice access expression is as
follows:

// Added
${syntax: sliceAccessExpression}

==== Type checking

// Added
After type checking, a bitslice access expression is represented as:

// Added
${syntax: sliceAccessExpressionIR}

// Added
${rulegroup-source: Expr_ok/sliceAccessExpression}
${rulegroup-prose: Expr_ok/sliceAccessExpression}

==== Local compile-time evaluation

// Added
${rulegroup-source: Expr_eval_lctk/sliceAccessExpressionIR}
${rulegroup-prose: Expr_eval_lctk/sliceAccessExpressionIR}

==== Compile-time evaluation

// Added
${rulegroup-source: Expr_inst/sliceAccessExpressionIR}
${rulegroup-prose: Expr_inst/sliceAccessExpressionIR}

==== Runtime evaluation

// Added
${rulegroup-source: Expr_eval/sliceAccessExpressionIR}
${rulegroup-prose: Expr_eval/sliceAccessExpressionIR}
