=== Literal expressions

// Added
Literal expressions denote fixed values of boolean, numeric, or string types:

// Added
${syntax: literalExpression booleanLiteral integerLiteral stringLiteral}

==== Type checking

// Added
After type checking, literal expressions have the form:

// Added
${syntax: literalExpressionIR}

// Added
${rulegroup-source: Expr_ok/literalExpression}
${rulegroup-prose: Expr_ok/literalExpression}
