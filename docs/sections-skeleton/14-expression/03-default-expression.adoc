[#sec-default-expression]
=== Default expressions

Default expressions provide a way to specify a default value for a type:

${syntax: defaultExpression}

See <<sec-default>> for more details on default types. When default expressions
are cast to a specific type, they evaluate to the default value for that type.
See <<sec-default-value>> for more details on default values.

==== Type checking

After type checking, default expressions have the form:

${syntax: defaultExpressionIR}

${rulegroup-source: Expr_ok/defaultExpression}
${rulegroup-prose: Expr_ok/defaultExpression}

==== Local compile-time evaluation

${rulegroup-source: Expr_eval_lctk/defaultExpressionIR}
${rulegroup-prose: Expr_eval_lctk/defaultExpressionIR}

==== Compile-time evaluation

${rulegroup-source: Expr_inst/defaultExpressionIR}
${rulegroup-prose: Expr_inst/defaultExpressionIR}

==== Runtime evaluation

${rulegroup-source: Expr_eval/defaultExpressionIR}
${rulegroup-prose: Expr_eval/defaultExpressionIR}
