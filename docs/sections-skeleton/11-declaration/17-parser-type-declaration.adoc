=== Parser type declaration

// Added
A parser type declaration introduces a new parser type.

// Added
${syntax: parserTypeDeclaration}

// Added
See <<sec-parser-object-type>> for more details on parser object types.

==== Type checking

// Added
After type checking, a parser type declaration has the following form:

// Added
${syntax: parserTypeDeclarationIR}

// Added
${rulegroup-source: Decl_ok/parserTypeDeclaration}
${rulegroup-prose: Decl_ok/parserTypeDeclaration}

==== Compile-time evaluation

// Added
At compile-time, a parser type declaration is loaded into the context with:

// Added
${rulegroup-source: Decl_inst/parserTypeDeclarationIR}
${rulegroup-prose: Decl_inst/parserTypeDeclarationIR}
