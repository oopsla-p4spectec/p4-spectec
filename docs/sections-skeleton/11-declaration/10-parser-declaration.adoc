[#sec-parser-declaration]
=== Parser declarations

// Added
A `parser` declaration introduces a programmable parser:

// Added
${syntax: parserDeclaration parserLocalDeclaration parserState}

==== Type checking

// Added
After type checking, a parser declaration is represented in P4~IR~ as:

// Added
${syntax: parserDeclarationIR parserLocalDeclarationIR parserStateIR}

// Added
${rulegroup-source: Decl_ok/parserDeclaration}
${rulegroup-prose: Decl_ok/parserDeclaration}

==== Compile-time evaluation

// Added
At compile-time, parser declarations are loaded into the context as parser
object constructors.

// Added
${rulegroup-source: Decl_inst/parserDeclarationIR}
${rulegroup-prose: Decl_inst/parserDeclarationIR}

// Added
When instantiated, an instance of a parser declaration is created:

// Added
${syntax: parserObject}

// Added
See <<sec-parser-object>> for details about parser objects.
