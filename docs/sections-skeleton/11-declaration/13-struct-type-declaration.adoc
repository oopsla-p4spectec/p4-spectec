[#sec-struct-type-declaration]
=== Struct type declaration

// Added
A struct type is defined with the following syntax:

// Added
${syntax: structTypeDeclaration typeFieldList typeField}

// Added
See <<sec-struct>> for more information about struct types.

==== Type checking

// Added
After type checking, a struct type declaration is represented in P4~IR~ as:

// Added
${syntax: structTypeDeclarationIR typeFieldListIR typeFieldIR}

// Added
${rulegroup-source: Decl_ok/structTypeDeclaration}
${rulegroup-prose: Decl_ok/structTypeDeclaration}

==== Compile-time evaluation

// Added
At compile-time, struct type declarations are loaded into the context by:

// Added
${rulegroup-source: Decl_inst/structTypeDeclarationIR}
${rulegroup-prose: Decl_inst/structTypeDeclarationIR}
