=== Match kind declarations

// Added
${syntax: matchKindDeclaration}

// Added
Similar to ``error`` declarations, ``match_kind`` declarations introduce
match kinds. See <<sec-match-kind>> for more details on match kinds.

==== Type checking

// Added

${rulegroup-prose: Decl_ok/matchKindDeclaration}
