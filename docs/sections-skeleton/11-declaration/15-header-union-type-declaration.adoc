[#sec-header-union-type-declaration]
=== Header union type declaration

// Added
A header union type is defined with the following syntax:

// Added
${syntax: headerUnionTypeDeclaration typeFieldList typeField}

// Added
See <<sec-header-union>> for more information about header union types.

==== Type checking

// Added
After type checking, a header union type declaration is represented in P4~IR~ as:

// Added
${syntax: headerUnionTypeDeclarationIR typeFieldListIR typeFieldIR}

// Added
${rulegroup-source: Decl_ok/headerUnionTypeDeclaration}
${rulegroup-prose: Decl_ok/headerUnionTypeDeclaration}

==== Compile-time evaluation

// Added
At compile-time, header union type declarations are loaded into the context by:

// Added
${rulegroup-source: Decl_inst/headerUnionTypeDeclarationIR}
${rulegroup-prose: Decl_inst/headerUnionTypeDeclarationIR}
