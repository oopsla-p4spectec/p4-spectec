[#sec-package-type-declaration]
=== Package type declaration

// Added
A package type declaration introduces a new package type.

// Added
${syntax: packageTypeDeclaration}

// Added
See <<sec-package-object-type>> for more details on package object types.

==== Type checking

// Added
After type checking, a package type declaration has the following form:

// Added
${syntax: packageTypeDeclarationIR}

// Added
${rulegroup-source: Decl_ok/packageTypeDeclaration}
${rulegroup-prose: Decl_ok/packageTypeDeclaration}

==== Compile-time evaluation

// Added
At compile-time, a package type declaration is loaded into the context with:

// Added
${rulegroup-source: Decl_inst/packageTypeDeclarationIR}
${rulegroup-prose: Decl_inst/packageTypeDeclarationIR}
