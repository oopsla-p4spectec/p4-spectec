[#sec-header-type-declaration]
=== Header type declaration

// Added
A header type is defined with the following syntax:

// Added
${syntax: headerTypeDeclaration typeFieldList typeField}

// Added
See <<sec-header>> for more information about header types.

==== Type checking

// Added
After type checking, a header type declaration is represented in P4~IR~ as:

// Added
${syntax: headerTypeDeclarationIR typeFieldListIR typeFieldIR}

// Added
${rulegroup-source: Decl_ok/headerTypeDeclaration}
${rulegroup-prose: Decl_ok/headerTypeDeclaration}

==== Compile-time evaluation

// Added
At compile-time, header type declarations are loaded into the context by:

// Added
${rulegroup-source: Decl_inst/headerTypeDeclarationIR}
${rulegroup-prose: Decl_inst/headerTypeDeclarationIR}
