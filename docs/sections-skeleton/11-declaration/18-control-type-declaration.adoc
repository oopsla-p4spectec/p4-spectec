=== Control type declaration

// Added
A control type declaration introduces a new control type.

// Added
${syntax: controlTypeDeclaration}

// Added
See <<sec-control-object-type>> for more details on control object types.

==== Type checking

// Added
After type checking, a control type declaration has the following form:

// Added
${syntax: controlTypeDeclarationIR}

// Added
${rulegroup-source: Decl_ok/controlTypeDeclaration}
${rulegroup-prose: Decl_ok/controlTypeDeclaration}

==== Compile-time evaluation

// Added
At compile-time, a control type declaration is loaded into the context with:

// Added
${rulegroup-source: Decl_inst/controlTypeDeclarationIR}
${rulegroup-prose: Decl_inst/controlTypeDeclarationIR}
