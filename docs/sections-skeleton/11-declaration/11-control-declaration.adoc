[#sec-control-declaration]
=== Control declarations

// Added
A `control` declaration introduces a control block:

// Added
${syntax: controlDeclaration controlLocalDeclaration controlBody}

==== Type checking

// Added
After type checking, a control declaration is represented in P4~IR~ as:

// Added
${syntax: controlDeclarationIR controlLocalDeclarationIR controlBodyIR}

// Added
${rulegroup-source: Decl_ok/controlDeclaration}
${rulegroup-prose: Decl_ok/controlDeclaration}

==== Compile-time evaluation

// Added
At compile-time, control declarations are loaded into the context as control
object constructors.

// Added
${rulegroup-source: Decl_inst/controlDeclarationIR}
${rulegroup-prose: Decl_inst/controlDeclarationIR}

// Added
When instantiated, an instance of a control declaration is created:

// Added
${syntax: controlObject}

// Added
See <<sec-control-object>> for details about control objects.
